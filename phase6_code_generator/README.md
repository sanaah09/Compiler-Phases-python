# Phase 6: Code Generator

This phase implements Target Code Generation, which is the final phase of a compiler.

The code generator converts optimized intermediate code into assembly-like
target code using simple instructions such as MOV, ADD, and MUL.

Language Used:
Python

Files in This Folder:
- codegen.py – Python program for target code generation
- codegen_input.txt – Optimized intermediate code input
- codegen_output.txt – Generated target code
- README.md – Documentation for Phase 6

How to Run:
1. Open a terminal in this folder.
2. Run the command:
   python3 codegen.py
3. The target code will be generated in codegen_output.txt
